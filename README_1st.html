CrNiCKL (pronounced chronicle) is a database for time series running on top of SQL and (soon) NoSQL systems.

There are various resources available for learning about CrNiCKL. The first is the javadoc documentation, which can be generated from the source code, or which can be browsed at the project website.

Another resource is the demo released with source code on the project website.

Finally there is my blog where you will find discussions about time series software.
