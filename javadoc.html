CrNiCKL (pronounced chronicle) provides a data model and a database for generic time series. It uses the Time2 library. The software does not depend on SQL, can run on top of NoSQL databases, and is extensible. A JDBC implementation is provided.

More information on CrNiCKL and the Time2 library can be found on the project website.

The current version of the software is 1.1.0, built on July 17, 2012.
